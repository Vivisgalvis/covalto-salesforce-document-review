# Salesforce Document Review Challenge

## Overview

Solution developed for the Covalto Salesforce Technical Challenge.

Features:

- PDF document visualization
- Document validation checklist
- Review comments history
- Document status tracking
- Email notifications
- Lightning Web Component (LWC)
- Apex backend services

## Technologies

- Salesforce
- Lightning Web Components
- Apex
- Salesforce Files
- Lightning App Builder

## Data Model

Lead
│
└── Document_Review__c
│
└── Review_Comment__c
